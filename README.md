# JSCalc, a simple calculator app!

This is a basic calculator made with HTML, CSS and JavaScript.

## What's working:
- Addition
- Substraction
- Multiply
- Division
- Remainder
- Plus/minus
- Clear

## Bugs:
- Clicking an operator after clearing displays "undefined".
- Large numbers go over the screen.
