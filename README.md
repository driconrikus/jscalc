# JSCalc 1.0, a simple calculator web app!

This is a basic calculator made with HTML, CSS and JavaScript, it also works with your keyboard!

## What's working:
- Addition
- Substraction
- Multiply
- Division
- Remainder
- Plus/minus
- Clear
- Keyboard input

## Bugs:
- Extremely large numbers will cause the equals sign to disappear.
